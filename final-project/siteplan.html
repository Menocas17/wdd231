Site Name: Tasty Eats

This name represents a web page that is focused in helping you find the ideal tasty recipe for the any ocassion.
Optional domain availabitily: tastyrepcipies.com

Site Purpose:

This website will provide an extense recepies catalog, that will be sorted by category, country and also you will be able to search for any specific recipe or add your owns recipies and saved them in the website.

Scenarios:

Where I can find a variety of recipies to chose from, I do not know what to cook?

It is possible to look for a recipe by the ingridients I have available at my home?

Can I save my own recipies online, so I can access to them later?

Color Schema:

  • Green = #88CB27, it will be used as the color of the logo and also as main color for the buttons
  • Blue = #13548f, it will be used for the ham manu, as secondary color for the buttons and it could be use for some headings and as a contrast color.
  • Beige = #ffffd8af, it will be use for the background color in some pages of the website like the home page

Typography:

- Roboto

This font is going to be the general font used in the body, this sitplan uses the font as an example

- Poppins

This font is going to be used in the nav menu, buttons, some headings and labels

-Pacifico

This fonts will be used in headings and in the logo as the main font
